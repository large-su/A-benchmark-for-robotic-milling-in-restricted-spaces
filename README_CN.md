# 受限空间机器人铣削基准测试

面向受限空间机器人铣削场景的标准化姿态规划算法评估基准任务。

## 概述

在加工大型复杂构件时，工业机器人常需在受限空间中作业——例如螺旋桨叶根重叠区域、涡轮叶片通道以及航天舱体内壁等——这对传统姿态规划方法提出了严峻挑战。本仓库提供了一个**公开可用、可复现的基准测试几何模型**，用于在典型受限空间工况下系统性地评估和比较机器人铣削姿态规划算法。

该基准设计为一个**不规则半封闭工作空间**，内部所有障碍物均为非对称倾斜结构，迫使机器人在整个加工过程中持续调整多维铣削姿态。值得注意的是，传统单维度（仅冗余角优化）规划策略在此场景下**无法获得可行轨迹**，从而验证了多维姿态规划的必要性。

## 基准参数

| 参数 | 数值 |
|------|------|
| 受限空间尺寸（不含辅助支撑） | 1.30 m × 0.96 m × 1.03 m |
| 加工轨迹 | 1.3 m 长水平直线路径 |
| 离散刀触点间距 | 2 mm |
| 障碍物构型 | 全倾斜特征的不规则半封闭区域 |
| 机器人系统（原研究所用） | ABB IRB-6660 |
| 铣削主轴（原研究所用） | HITECO |
| 刀具（原研究所用） | 圆角铣刀，刀具半径 R = 12.5 mm，圆角半径 r = 5 mm |

> **说明：** 虽然原始实验验证在 ABB IRB-6660 系统上完成，但本基准几何模型**不依赖于特定机器人型号**，可与任何六自由度工业机器人配合使用，用于姿态规划算法评估。

## 设计原理

基准几何模型基于以下设计原则构建：

- **代表性**：不规则半封闭结构参考了螺旋桨叶根重叠区和涡轮叶片通道等实际加工中常见的受限空间场景。
- **非对称障碍**：所有障碍物表面均为非对称倾斜设计，确保可行姿态解空间沿刀位轨迹动态变化，而非保持静态。
- **多方向约束**：机器人系统同时受到来自左、右、上方障碍物的限制，形成极度狭窄且动态变化的可行解空间。
- **可复现性**：几何模型提供多种标准 CAD 格式，支持不同研究团队和软件平台精确复现基准测试环境。

**推荐使用方式：** 将 `.STEP` 文件导入您的机器人仿真环境（如 RoboDK、MATLAB Robotics Toolbox、CoppeliaSim 或自定义仿真平台），作为姿态规划评估的障碍物几何模型。

## 使用方法

1. **搭建受限空间环境**：将障碍物几何文件导入仿真平台，并按照原始论文中定义的坐标系进行定位。
2. **定义加工任务**：在受限空间内布置一条 1.3 m 长的水平直线轨迹，离散刀触点间距为 2 mm（共 651 个刀触点）。
3. **运行姿态规划算法**：评估您的算法能否在整个加工过程中生成一条可行且连续的姿态轨迹，同时完全避开所有障碍物。
4. **评估与比较**：建议的评估维度包括：
   - **可行性**：算法能否找到无碰撞的姿态轨迹？
   - **平滑性**：规划轨迹的关节运动平滑程度如何？
   - **加工性能**：规划姿态的刚度、精度或稳定性特征如何？
   - **计算效率**：算法求解所需的计算时间是多少？
